# v1.1.1 リリースノート

## 概要
- 対象: iRIC_DataScope
- リリース日: 2025-12-26
- 目的: X-Y分布画像出力（XY Value Map）のUI改善と起動/配布手順の整理

## 主な変更点
### 追加
- XY Value Map のカラーマップ選択を3種類に拡張（最小/最大の2色、虹色（iRIC風）、色相回転）
- 出力ステップを範囲スライダー＋開始/終了入力で指定できるように変更
- app.py から各ツールを呼び出すための launcher 入口を追加（各機能側）

### 改善
- XY Value Map の設定UIを整理（色/スケール統合、表示・フォントサイズ・出力の分割）
- 表示/出力オプションの説明を追加し、意図が分かる文言に調整
- 起動入口を app.py 基準に統一（`iRIC_DataScope/__main__.py` 追加）

### 修正
- 虹色（iRIC風）選択時に最小/最大色が無効になることをUIで明示
- PyInstaller 用の spec を app.py 入口基準に整理（最小構成化）

## 影響範囲
- 影響を受ける機能: XY Value Map のUI/出力オプション、ランチャー起動経路
- 互換性: 既存の出力ファイル仕様は維持（UI操作方法のみ変更）

## 既知の問題
- 最小構成の PyInstaller ビルドでは、環境によって特定機能の依存不足が発生する可能性あり

## 使い方 / 更新手順
- 既存ユーザは通常通り起動即可（設定UIの並び・名称が更新されています）
- PyInstaller ビルドは `iRIC_DataScope.spec` を使用
