# v1.1.3 リリースノート

## 概要
- 対象: iRIC_DataScope
- リリース日: 2026/01/19
- 目的: 起動体験の改善（スプラッシュ）とヘルプ導線の統一、ドキュメント拡充

## ユーザ向けの主な変更
- ランチャー起動時のスプラッシュ表示を改善し、起動中の待機が分かりやすくなりました。
- 各ツールのヘルプメニューが GitHub Pages の該当ドキュメントを開くように統一されました。
- ランチャーの入力パス判定を強化し、無効な入力の場合は起動ボタンが有効にならないようになりました。
- X-Y 分布画像出力で、初期読み込み完了後のUI更新時にウィンドウが閉じられている場合の安全策を追加しました。

## ドキュメント
- ユーザ向けドキュメントを全体的に整理し、最短手順や注意事項、用語補足を追加しました。
- GUIのスクリーンショットや出力例画像を更新・追加しました。
- GitHub Pages / MkDocs 設定を見直し、公開ナビゲーションと除外ドキュメントを整理しました。

## ビルド / 配布
- PyInstaller のエントリポイントを `main.py` に変更し、スプラッシュを組み込みました。
- 出力名を `iRIC_DataScope-v1.1.3` に更新しました。

## 影響範囲
- 影響を受ける機能: ランチャー、横断重ね合わせ図作成、時系列データ抽出、X-Y 分布画像出力
- 互換性: 既存の操作手順と出力形式は維持されます。
