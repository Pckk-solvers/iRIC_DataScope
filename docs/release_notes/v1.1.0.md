# v1.1.0 リリースノート

## 概要
- 対象: iRIC_DataScope
- リリース日: 2025-12-XX
- 目的: X-Y分布画像出力（XY Value Map）の機能拡充と基盤整理

## 主な変更点
### 追加
- X-Y分布画像出力（XY Value Map）の機能拡充（ROI/プレビュー/出力オプション）
- カラーマップ設定の追加と整理（最小/最大色ベースのグラデーション）
- 出力ステップ範囲と間引き設定を追加

### 改善
- プレビューと出力の見た目を一致させる調整
- 出力画像の余白・サイズ調整の改善
- 日本語フォントの利用を強化
- 入力経路の整理（プロジェクトフォルダ/ipro対応の強化）

### 修正
- カラーバーラベルが出力に反映されない問題を修正
- GUI/処理のリファクタリングにより構造を整理

## 影響範囲
- 影響を受ける機能: XY Value Map（X-Y分布画像出力）、入力処理（CGNS/ipro）
- 互換性: 既存の出力フォーマットは維持

## 既知の問題
- 特になし

## 使い方 / 更新手順
- 既存ユーザは通常通り起動即可
- 新規機能はランチャーから「X-Y分布画像出力」を選択
