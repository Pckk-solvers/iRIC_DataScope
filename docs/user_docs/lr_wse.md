# 左右岸水位抽出

## 目的
設定（KP と左右岸の I/J 指定）に基づいて、左右岸の水位等を抽出し、KPごとの最大水位をExcelに整理して出力します。

## 入力
- 入力（どれか1つ）
  - プロジェクトフォルダ
  - `.ipro`
  - `Result_*.csv` が格納されたCSVフォルダ
- 設定ファイル（必須）
  - `setting.csv`（GUIから選択）

## 出力
- Excel（例: `LR_WSE.xlsx`）
- オプションで「中間CSVを出力する」をONにすると、出力フォルダ配下に `Excel名` のサブフォルダを作成して中間CSVを出力します。

## 使い方（最短）
1. 入力（プロジェクトフォルダ / `.ipro` / CSVフォルダ）を指定
2. 設定ファイル（CSV）を指定
3. 出力フォルダと出力ファイル名（Excel）を指定
4. 実行

## 注意
- 設定側の `LI/LJ/RI/RJ` は数値として解釈できる形式である必要があります（例: `12`, `12.0`）。
- 入力データに `watersurfaceelevation(m)` / `elevation(m)` が存在しない場合、抽出結果が欠損になります。
