# X-Y Value マップ

## 目的
メッシュ上の `X, Y` と選択した変数（Value）を使って、ステップごとの分布を画像として出力します。

## 入力
- 入力（どれか1つ）
  - プロジェクトフォルダ
  - `.ipro`
  - `Result_*.csv` が格納されたCSVフォルダ

## 出力
- 画像ファイル（複数ステップ分）

## 使い方（概要）
1. 入力と出力フォルダを指定
2. Value（列）を選択
3. 表示範囲（ROI）を指定（必要に応じて）
4. プレビューで見た目を確認
5. 出力（全ステップ / プレビュー中ステップ）

## 補足
- プレビューは負荷を下げるため、表示処理を軽量化しています（点数が多い場合は特に）。
- 境界データは対象外です。
