# 横断重ね合わせ図作成

## 目的
指定した条件で横断方向のプロファイルを作成し、複数ステップの結果を重ね合わせて可視化します。

## 入力
- 入力（どれか1つ）
  - プロジェクトフォルダ
  - `.ipro`
  - `Result_*.csv` が格納されたCSVフォルダ

## 出力
- 図（画像ファイル）または関連ファイル（機能の画面に従う）

## 使い方（概要）
1. 入力と出力フォルダを指定
2. モードや対象ステップ、変数などを選択
3. 実行して出力を確認

## 補足
- CSVフォルダ入力の場合、必要な列が `Result_*.csv` に存在する必要があります。
