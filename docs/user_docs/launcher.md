# ランチャー（起動ウィンドウ）

## 目的
入力（プロジェクトフォルダ / `.ipro` / CSVフォルダ）と出力フォルダを指定し、各機能（別ウィンドウ）を起動します。

![ランチャー](../images/launcher.png)

## 入力
- **プロジェクトフォルダ**（推奨）
  - 例: `Case1.cgn` / `Solution*.cgn` が含まれるフォルダ
- **`.ipro`**
  - iRIC のプロジェクトファイル
- **CSVフォルダ**
  - iRIC 出力の `Result_*.csv` が入っているフォルダ

## 出力
- 出力フォルダは、各機能が生成するファイルの保存先として使われます。

## 使い方
1. 「入力パス」でプロジェクトフォルダ / `.ipro` / CSVフォルダを選択
2. 「出力フォルダ」で保存先を選択
3. 実行したい機能ボタンを押す

## 補足
- ランチャーは、入力を自動変換（CSV化）してから各機能を起動する方式ではなく、各機能が入力を直接読み込む方式です。
- 詳細手順は「ヘルプ → マニュアルを開く」を参照してください。
