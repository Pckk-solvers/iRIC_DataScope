# 時系列データ抽出

## 目的
指定した地点（または条件）に対して、複数ステップの値を集計して時系列データとして出力します。

## 入力
- 入力（どれか1つ）
  - プロジェクトフォルダ
  - `.ipro`
  - `Result_*.csv` が格納されたCSVフォルダ

## 出力
- 集計結果（CSV など。機能の画面に従う）

## 使い方（概要）
1. 入力と出力フォルダを指定
2. 抽出したい変数（列）を選択
3. 実行して出力を確認

## 補足
- 変数一覧は、入力がプロジェクトフォルダ / `.ipro` の場合はCGNSから、CSVフォルダの場合はCSVの列から取得します。
